# toolbox
# 我的工具类软件

一个多功能工具类软件，旨在帮助用户提高工作效率，提供多种实用工具，包括但不限于股票跟踪、历史数据查询、新闻摘要等。

## 功能

- **股票功能**：获取当前股票的股价和历史数据，展示最新相关的股票新闻。
- **新闻摘要**：自动分析并总结最新股票相关新闻。
- **其他工具**：更多功能持续开发中。
